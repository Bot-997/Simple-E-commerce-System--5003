# **E-commerce Backend System**

A simplified e-commerce backend system developed for a student project. 
It is built using Java Spring Boot and MyBatis-Plus, strictly following RESTful API design standards. 
The system covers core e-commerce functionalities including user management, product browsing (SKU support), shopping cart, wishlist, and order processing.

####  Technology Stack

1. [x] Language: Java 17
2. [x] Framework: Spring Boot 3.4.12
3. [x] ORM: MyBatis-Plus 3.5.7
4. [x] Database: MySQL 8.0
5. [x] Build Tool: Maven
6. [x] Utilities: Lombok


#### Installation & Setup
1. Prerequisites：
###### JDK 17 or higher

###### MySQL 8.0

###### Maven 3.6+

###### IDE (IntelliJ IDEA / Eclipse)


3. Application Configuration
######  Open src/main/resources/application.properties and update your database credentials

4. Run / 运行项目
###### Run the application using Maven or your IDE

The server will start at: http://localhost:8080

Cross-Origin Resource Sharing is enabled by default in config/CorsConfig.java to support frontend development on a different port.
